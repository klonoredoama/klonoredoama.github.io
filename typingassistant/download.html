TypingAssistant v0.1.1

Automated system-wide accented character replacement.

Recommended

Standard Installer

The easiest way to use TypingAssistant. Downloads the latest version and sets up automatic background updates via updater.exe.

Portable (.zip)

Standalone files for advanced users. No installation required. Note: Does not support automatic updates.

Getting Started

1. Installation: Run the installer to set up the app in your local AppData folder.

2. Usage: Type any unaccented word and press the insert key to replace it with the accented version.

3. Configuration: Custom settings like trigger keys and delays are managed in config.json.
