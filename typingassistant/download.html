How to Use (Standard Installer)

  • Download and run the installer.exe.
  • The installer will automatically fetch the necessary files and create a Desktop Shortcut for you.
  • Run the app from the shortcut; this will open a new console window.
  • Once running, you may minimize the window; it will continue running system-wide.
  • Type any word unaccented and press the [insert] key (or your configured accept_key) to replace it with the accented word.
  • Updates are automatic: Every time you launch the app, it checks for updates and installs them using the included updater.exe.

How to Use (Portable / No-Install)

  • Download and extract the .zip file to a folder of your choice.
  • Run the typing_assistant.exe file; this will open a new console window.
  • Once running, you may minimize the window; it will continue running system-wide.
  • Type any word unaccented and press the [insert] key (or your configured accept_key) to replace it with the accented word.

Note: This version does not support automatic updates; you must download the new .zip manually to update.

Editing Settings

You can edit settings using the config.json file found in the application folder. Just open it, change the values, and save it to customize:

accept_key: Change the trigger key (default is insert).
delays: Adjust how fast the app deletes and types characters.
logging: Turn the app.log file on or off.
Notes: Please note that this app is still in its early stages. Ensure that config.json remains in the same folder as the .exe file for the app to function correctly.
